Change log 
(in-progress):
  * Refactored into a proper library: core logic moved into libmodemu2k;
    public API exposed via modemu2k.h at the project root
  * Removed gettext/NLS support and translations
  * Removed valgrind test setup
  * Expanded test suite: timeval, sockbuf, atcmd unit tests; IPv4/IPv6
    connection tests (IPv6 skipped with exit 77 if no interface present)
  * Fix connecting via hostname when first resolved address fails (e.g.
    localhost resolving to ::1 before 127.0.0.1) (issue #44)
  * Add listen mode (-l/--listen <port>) that accepts a single incoming
    TCP connection and uses it as the TTY; intended for use as a plugin
    with dosemu2 (issue #32)
  * Print a short usage hint and exit when invoked with no arguments;
    standalone stdin/stdout command mode is still reachable via -e (or
    any other flag)
  * Wrap modemu2k.h declarations in extern "C" guards for C++ consumers
