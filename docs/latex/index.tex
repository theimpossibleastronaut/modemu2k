modemu2k is a fork of modemu, originally developed by Toru Egashira

(Build status \href{https://github.com/theimpossibleastronaut/modemu2k/actions/workflows/c-cpp.yml}{\texttt{ https\+://github.\+com/theimpossibleastronaut/modemu2k/actions/workflows/c-\/cpp.\+yml}})\hypertarget{index_autotoc_md0}{}\doxysection{What is modemu2k?}\label{index_autotoc_md0}
modemu2k adds telnet capability to a comm program. It can redirect telnet I/O to a pty so that a comm program can handle the pty as a tty with a real modem, and allows you to use a comm program\textquotesingle{}s scripting and file transfer features over telnet. Now supports IPv6 connections.

Note\+: while in the program if backspace doesn\textquotesingle{}t work, use CTRL+H.

1) Stand alone usage

Invoked with no option, \begin{DoxyVerb}modemu2k
\end{DoxyVerb}
 modemu2k reads from standard input and writes to standard output. Input

\begin{quote}
atd"{}localhost \mbox{[}port\mbox{]} \end{quote}
(prompt \char`\"{}$>$\char`\"{} is not shown) You will see your host\textquotesingle{}s login prompt (if a server is running). When you disconnect, you will get \char`\"{}\+NO   CARRIER\char`\"{} indication. Then input

\begin{quote}
atq \end{quote}
to quit modemu2k.

2) With a comm program

(This example uses minicom as the comm program)

Invoke with \char`\"{}-\/c\char`\"{} option, \begin{DoxyVerb}modemu2k -e "AT%B0=1%B1=1&W" -c "minicom -l -tansi -con -p %s"
\end{DoxyVerb}
 From within the comm program, if you have a server running, to connect you can enter\+: \begin{DoxyVerb}atd"localhost
\end{DoxyVerb}
 You could also connect to a BBS. For some telnet addresses, see\+:


\begin{DoxyItemize}
\item $<$$<$Prism BBS\href{http://filegate.net/}{\texttt{ http\+://filegate.\+net/}}
\item The Rusty Mailbox (telnet to trmb.\+ca, port 2030)
\item Synchronet BBS List\href{http://www.synchro.net/sbbslist.html}{\texttt{ http\+://www.\+synchro.\+net/sbbslist.\+html}}
\end{DoxyItemize}

To quit modemu2k, just quit the comm program.

A script to invoke minicom as mentioned above will be installed to your bin directory when {\ttfamily make install} is run.\hypertarget{index_autotoc_md4}{}\doxysection{Escaping to command mode and returning}\label{index_autotoc_md4}
